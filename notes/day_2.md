# Day 2 – PDF Extraction

## Objective

Extract text from PDF files and save the extracted content for further processing.

## Concepts Learned

* PDF Documents
* Pages in PDFs
* Text Extraction
* PyMuPDF
* PDF to Text Conversion

## Tasks Completed

1. Installed PyMuPDF.
2. Created pdf_extraction.ipynb.
3. Added sample PDF report.
4. Opened PDF using fitz.
5. Checked total pages.
6. Extracted text from first page.
7. Extracted text from all pages.
8. Combined extracted text.
9. Counted characters.
10. Counted words.
11. Saved extracted text to report.txt.
12. Verified extracted output.

## Workflow

PDF
↓
PyMuPDF
↓
Extract Text
↓
Save report.txt

## Output Files

data/raw/report.pdf

artifacts/extracted_text/report.txt

## Key Learnings

* LLMs cannot directly understand PDFs.
* PDFs must first be converted to text.
* Text extraction is the first stage of any document-based AI application.
